Adds scrolling functionality to the Sharrre bar built into the Hueman template

== Description ==

Can only be used with the Hueman template. Template must include share functionality from the hueman site <http://alxmedia.se/themes/hueman/>.

This plugin will make the built in sharrre bar scrollable. It will start and stop at the top and bottom of the post entry block.

It also supports the responsiveness of the template.


* Update *
If you have installed of the Hueman template v2.1.1 or later, you do not need to install this plugin. This functionality has been ported into the theme. It has been turned off by default but can be turned on through the theme options under “blog”.

== Installation ==

1. Upload `Heman Scrollable Sharrre Bar` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Update the default settings as needed.
